# 🚀 Project Overview

The **Online Voting System with Face Detection** is a secure web-based voting platform developed to ensure transparent and fraud-free elections using facial recognition technology.

This system verifies voters using face detection before allowing them to cast their vote, helping prevent impersonation and duplicate voting.

---

# ✨ Features

✅ Secure Voter Registration  
✅ Face Detection Authentication  
✅ Admin Dashboard  
✅ Candidate Management  
✅ One Vote Per User Policy  
✅ Real-Time Vote Counting  
✅ Responsive User Interface  
✅ Secure Login System  
✅ Election Monitoring & Result Management  

---

# 🤖 Face Recognition Module

The project uses **OpenCV** for:
- Face Detection
- Facial Feature Extraction
- Voter Verification
- Fraud Prevention

This ensures that only verified users can participate in the voting process.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/online-voting-system.git
```

---

## 2️⃣ Open Project Folder

```bash
cd online-voting-system
```

---

## 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Server

```bash
python manage.py runserver
```
